# Data Sources

This lab uses public biomedical data sources only unless explicitly documented otherwise.

## Build 1 allowed sources

| Source | Purpose |
|---|---|
| DailyMed | Public drug-label sections |
| RxNorm / RxNav | Drug-name normalization |
| PubMed / NCBI E-utilities | Biomedical literature abstracts |

## Out of scope

The lab must not use:

- private patient data
- EHR exports
- claims data
- proprietary pharma data
- confidential company information
- commercial drug databases
- restricted internal company documents

## Data handling rule

All data used in this repo must be:

- public
- synthetic
- clearly documented
- safe to publish
- reproducible by another engineer
